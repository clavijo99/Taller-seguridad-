# Hints

## Nivel 1 (General)
- Revisa cómo se manejan los datos de usuario y los tokens.
- Observa cómo se procesan los formularios y las respuestas del backend.
- Analiza la configuración de seguridad en el backend y frontend.

## Nivel 2 (Más específico)
- ¿Las contraseñas están protegidas adecuadamente?
- ¿Qué ocurre si envías datos inesperados en los formularios?
- ¿El backend valida los datos recibidos?
- ¿Qué información devuelve el endpoint de perfil?

## Nivel 3 (Casi solución)
- Prueba inyectar SQL en los campos de login o post.
- Intenta insertar scripts en los comentarios o posts.
- Observa si el token JWT expira o puede ser manipulado.
- Revisa los encabezados CORS y los orígenes permitidos.
